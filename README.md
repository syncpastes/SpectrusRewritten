# SpectrusRewritten
Exploit for r/w to protected memory with lsass.exe
## Installation
1. Clone the repo
2. Extract the repo
3. Open in [Visual Studio 2015](https://visualstudio.microsoft.com/vs/older-downloads/)
4. Edit params in main to your choosing.
5. Run with target process.
